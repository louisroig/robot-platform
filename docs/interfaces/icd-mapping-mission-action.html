Action interface for triggering a complete drone mapping mission — arm, takeoff, capture, land.

Purpose

The /mapping/run_mission action is the control surface for the drone's autonomous mapping mission. It is invoked by mission_api when the operator taps "Map My Yard." It runs to completion over ~3 minutes, providing feedback at each state transition.

Action Definition

# Goal float32 altitude_m # target capture altitude AGL (default 30.0) string output_name # identifier for saved image, e.g. "front_yard_20260419" --- # Result bool success string mission_id # UUID of mission record string image_path # populated if success=true string failure_reason # populated if success=false --- # Feedback string phase # SM-DRN-001 state name: one of "DISARMED","ARMED","TAKEOFF","CLIMBING","CAPTURING","DESCENDING","LANDED","IMAGE_TRANSFER","GEOREF","ABORTING" float32 progress # 0.0 to 1.0 float32 altitude_m # current drone altitude float32 battery_pct # current drone battery
4.0

Semantics

On goal accept, the server returns an accepted response immediately (within 100 ms) and begins executing the mission. Progress feedback is published at ~2 Hz throughout. Goal can be canceled; cancellation triggers immediate land (ABORTING state). A successful result populates image_path; a failed result populates failure_reason.

Mission states and transitions are formally specified in SM-DRN-001.

Fault

FaultResponse
Arm failureResult: success=false, reason="arm_failed: <mavlink_error>". Drone never lifted.
GPS lock lost during missionMission aborts; drone lands at current position (ArduCopter failsafe).
Low battery during missionMission aborts; drone executes RTL (return to launch).
Operator cancellationSame as abort; drone lands.
Image capture failsDrone still lands; result carries success=false, reason="capture_failed".

Requirements

  • REQ-ICD-MAP-003-01
    Goal shall be accepted within 100 ms of submission when the drone is in a valid pre-arm state.
    verified by: TEST-MAP-020
  • REQ-ICD-MAP-003-02
    Feedback shall be emitted at ≥ 1 Hz throughout mission execution.
    verified by: TEST-MAP-021
  • REQ-ICD-MAP-003-03
    On cancellation or safety abort, the drone shall reach the ground within 30 s.
    verified by: TEST-MAP-022
