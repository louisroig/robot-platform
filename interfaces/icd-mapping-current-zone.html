Active zone polygon (outer boundary + exclusions). Latched topic; published whenever the active zone changes.

1.0

Purpose

Carries the currently-active operational zone — the outer boundary polygon the rover may occupy plus any exclusion polygons marking no-go areas within it. Consumed by the geofence enforcer to compute /geofence/state.

2.0

Participants

boundary_ingester
SRSSRS-MAP-004SourcePersisted zone on disk; published on load and on active-zone change.
Subscribers

geofence_enforcerUses polygon for point-in-polygon tests.telemetry_publisherForwards polygon metadata to app for display.
3.0

Message

Type: platform_msgs/msg/ZoneBoundary.

std_msgs/Header header string name geometry_msgs/Polygon outer geometry_msgs/Polygon[] exclusions # no-go polygons inside outer

Each polygon's points are in the map frame (meters). Converted from lat/lon by the server at zone-load time using the active aerial image meta.

4.0

Semantics

A single active zone is in effect at any time. Changing zones is accomplished by publishing a new message on this topic; subscribers immediately switch to the new polygon.

Outer polygon must be simple (non-self-intersecting), counter-clockwise-oriented, with at least 3 vertices. Exclusions must be fully contained within outer, simple, and non-overlapping. These constraints are enforced by boundary_ingester; a message on this topic always satisfies them.

5.0

Timing

Published on-change only. Typical cadence: a few times per day at most (operator selects zones). No regular keepalive.

6.0

QoS

RELIABLE, KEEP_LAST depth 1, TRANSIENT_LOCAL — the currently-active zone persists for late subscribers.

7.0

Fault

FaultResponse
No zone loadedTopic has never published. geofence_enforcer operates in "no geofence" mode (INSIDE, infinite distance). safety_monitor does not treat this as a fault — the operator may be teleoperating without a zone.
Malformed polygonCaught at boundary_ingester; never reaches this topic.
Late subscriptionSubscriber immediately receives the last-published zone via TRANSIENT_LOCAL durability.
