1.0

Purpose

The roboclaw_driver is the vendor-boundary node for the RoboClaw 2x15A motor controller. Upstream it exposes ROS-standard Twist and Odometry; downstream it speaks RoboClaw's binary serial protocol over USB. Above it, the system treats the motor controller as generic; below it, the driver handles all vendor specifics.

2.0

Scope

In scope. Serial communication with RoboClaw; Twist-to-per-track kinematics; encoder read and integration into odometry; battery voltage read and publication; command-timeout safe-halt.

Out of scope. Nav2 costmap, trajectory following, slip compensation (iter 3+), automatic chassis parameter estimation.

4.0

Functional Requirements

  • SRS-HAL-001-F01
    Convert every received Twist into per-track velocities via skid-steer kinematics using configured track width.
    verified by: TEST-HAL-001
  • SRS-HAL-001-F02
    Clip commanded velocities to configured max_linear_vel and max_angular_vel with sign preservation.
    verified by: TEST-HAL-012
  • SRS-HAL-001-F03
    Read encoder counts and battery voltage at 50 Hz and 1 Hz respectively; publish Odometry and BatteryState accordingly.
    verified by: ICD-HAL-003 tests
  • SRS-HAL-001-F04
    Halt motor commands if no /hal/cmd_vel_safe message has been received within cmd_vel_timeout_ms (default 500 ms).
    verified by: TEST-HAL-009
  • SRS-HAL-001-F05
    Reject Twist messages containing NaN or Inf in any field; emit diagnostic; hold last-known-good command or halt per policy.
    verified by: TEST-HAL-011
5.0

Performance Requirements

  • SRS-HAL-001-P01
    Serial round-trip per cycle ≤ 10 ms.
    verified by: TEST-HAL-013
  • SRS-HAL-001-P02
    CPU usage < 8% on Pi 5.
    verified by: TEST-HAL-014
6.0

Safety Requirements

  • SRS-HAL-001-S01
    Command timeout (F04) is the subscriber-side component of the two-layer safety: even if safety_monitor fails to publish zero, stale commands themselves cause halt.
    maps to: SR-005
  • SRS-HAL-001-S02
    On startup, motors shall be held at zero velocity until first valid /hal/cmd_vel_safe is received.
    maps to: SR-008
7.0
