Purpose

Captures the material parameter choices for the drone's flight controller firmware. The full parameter dump (~1100 parameters) is kept in deploy/firmware/arducopter_params.yaml; this document lists only the parameters we deliberately diverge from default on.

Nomenclature: despite the "ArduRover" hint in the filename for historical reasons, the iteration-1 drone is a quad running ArduCopter. The parameter set is ArduCopter's.
