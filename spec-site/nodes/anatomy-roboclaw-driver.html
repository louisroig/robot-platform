ROS 2 NODE · HAL
roboclaw_driver
Vendor-boundary node for the RoboClaw 2x15A motor controller. Translates Twist commands into skid-steer per-track velocities, integrates encoder counts into odometry, reports battery. Above this layer the motor controller is generic; below it, vendor-specific.
