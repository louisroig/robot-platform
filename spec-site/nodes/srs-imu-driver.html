imu_driver

Driver for the ISM330DHCX 6-DoF IMU plus the MMC5983MA magnetometer. Produces fused orientation and raw inertial/magnetic readings.

1.0

Purpose

Reads the ISM330DHCX accelerometer + gyroscope at 100 Hz and the MMC5983MA magnetometer at 50 Hz over I²C. Applies bias correction, hard/soft-iron calibration, and an on-board complementary Madgwick filter to produce orientation. Publishes both to /hal/imu/data and /hal/imu/mag.

2.0

Scope

In: I²C transactions, calibration load/apply, Madgwick fusion, staleness-safe publication, diagnostics. Out: physical calibration (one-time, separate tool); sensor replacement logic.

3.0

Interfaces

Publishers: /hal/imu/data (100 Hz), /hal/imu/mag (50 Hz), /diagnostics. No subscribers. No services in iteration 1 (a /imu/recalibrate service is deferred).

4.0

Functional Requirements

  • SRS-HAL-002-F01
    Sample the IMU at 100 Hz, the magnetometer at 50 Hz, each on its own timer; publish in REP-103 axis convention.
    verified by: TEST-HAL-020
  • SRS-HAL-002-F02
    Apply calibration matrices from imu_calibration.yaml; cease publication if the file is missing rather than emit raw values.
    verified by: TEST-HAL-026
  • SRS-HAL-002-F03
    Run a Madgwick fusion filter over gyro + accel; populate orientation and orientation_covariance.
    verified by: TEST-HAL-021
5.0

Performance

  • SRS-HAL-002-P01
    Orientation error at rest shall be < 2° pitch/roll after 10 s warmup.
    verified by: TEST-HAL-021
  • SRS-HAL-002-P02
    CPU usage < 5% on Pi 5.
    verified by: TEST-HAL-024
6.0

Safety

  • SRS-HAL-002-S01
    On I²C fault, node shall stop publishing rather than emit last-known values; safety_monitor's staleness logic then triggers ESTOP.
    maps to: SR-006
7.0

Dependencies

  • rclpy, sensor_msgs, tf_transformations.
  • smbus2 (I²C access).
  • Madgwick implementation (local module).

9.0

Failure Modes

  • Sensor NAK: retry 3× → halt.
  • Calibration missing: refuse to start, emit error log.
  • Implausible magnetic field (> 200 µT): warning diagnostic; continue publishing (down-weighted by EKF).
10.0
