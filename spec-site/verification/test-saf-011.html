ESTOP latency on person detection

Integration test: measure end-to-end latency from injected person detection to first zero-velocity message on /hal/cmd_vel_safe.

Test ID
TEST-SAF-011
Method
Integration
Automation
CI (launch_testing)
Revision
0.1 · draft
1.0

Requirements verified

  • SR-002 — cease motion within 500 ms of person within 3 m.
  • REQ-ICD-002-02 — zero within 50 ms of ESTOP entry.
  • FR-004 — detect and stop for humans/pets.
2.0

Method

Integration test under launch_testing. No physical hardware; the perception pipeline is mocked and a test harness injects a synthetic person detection. The safety stack runs under real DDS.

3.0

Preconditions

  • Full safety stack launched: safety_monitor, roboclaw_driver (serial mocked), estop_monitor (GPIO mocked), geofence_enforcer, living_thing_filter.
  • No zone loaded (geofence inactive).
  • safety_monitor has completed startup self-test; state = OK.
  • A publisher injecting teleop Twist at 0.3 m/s is running to keep cmd_vel_safe non-zero.
4.0

Procedure

  1. Start recording /perception/people and /hal/cmd_vel_safe via rosbag2.
  2. Wait 2 s for settling.
  3. Using the test publisher, emit a single DetectionArray on /perception/detections containing one detection with class=person, confidence=0.9, distance_m=2.0 (within danger radius of 3.0 m).
  4. Record the timestamp of publication (t_inject).
  5. Scan /hal/cmd_vel_safe for the first message with linear.x == 0 and angular.z == 0 after t_inject. Call its stamp t_zero.
  6. Record latency = t_zero − t_inject.
  7. Repeat steps 3–6 ten times.
  8. Compute min, max, mean, 95th percentile of latency.
5.0

Expected results

  • Every trial's latency < 50 ms.
  • p95 < 50 ms, mean < 30 ms.
  • No subsequent non-zero cmd_vel_safe messages until the synthetic detection is removed and dwell elapses.
6.0

Pass / fail criteria

Pass: All 10 trials latency ≤ 50 ms; no missed zeros; no spurious non-zeros during trial.

Fail: Any trial > 50 ms, OR any trial missing a zero within 200 ms, OR spurious non-zero before dwell expires.

7.0

Test data / fixtures

  • Fixture: test/fixtures/mock_person_detection.py — provides the synthetic DetectionArray publisher.
  • Fixture: test/fixtures/mock_serial_roboclaw.py — accepts serial commands and logs them (no hardware).
  • Fixture: test/fixtures/mock_gpio.py — emulates libgpiod, e-stop always released.
8.0

Automation status

Automated under launch_testing. Located at platform_safety/test/test_saf_011_estop_latency.py. Runs in CI on every PR touching platform_safety/*.
