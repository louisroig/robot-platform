How to read

Each requirement below carries:

  • A stable ID.
  • A statement using "shall."
  • Allocation — which node(s) implement it.
  • Verification trace — which test(s) prove it.
  • Rationale — why it exists, when non-obvious.

Requirements

  • FR-001
    The system shall map a user-specified yard by triggering an autonomous drone flight.
    Establishes the aerial-imagery foundation upon which boundary drawing depends.
    allocated to: drone_mission_coordinator, mission_api · verified by: VRF-FIELD
  • FR-002
    The user shall be able to draw polygonal boundaries on an aerial image from a mobile application.
    allocated to: mobile app, mission_api, boundary_ingester · verified by: VRF-FIELD steps 5–6
  • FR-003
    The rover shall respect drawn boundaries by ceasing motion when the boundary is crossed.
    Primary purpose of the geofence subsystem.
    allocated to: geofence_enforcer, safety_monitor · verified by: VRF-FIELD step 9
  • FR-004
    The rover shall detect humans and pets in its field of view and stop motion when detections occur within a configurable danger radius.
    allocated to: yolo_detector, living_thing_filter, safety_monitor · verified by: TEST-SAF-011
  • FR-005
    The user shall be able to teleoperate the rover from the mobile application.
    allocated to: mobile app, mission_api, twist_mux, safety_monitor, motor_driver · verified by: VRF-FIELD steps 7, 10
  • FR-006
    The rover shall provide live telemetry (pose, battery, safety state) to the mobile application.
    allocated to: telemetry_publisher · verified by: TEST-API-010
  • FR-007
    The system shall persist drawn boundaries across reboots.
    allocated to: boundary_ingester · verified by: TEST-MAP-001
  • FR-008
    The drone shall capture georeferenced aerial imagery suitable for boundary drawing.
    allocated to: drone_mission_coordinator, image_retriever, georeferencer · verified by: TEST-MAP-010, TEST-MAP-011
  • FR-009
    The system shall log all motion commands and safety events to persistent storage.
    Post-incident forensic capability; audit for stakeholder review.
    allocated to: rosbag2 (systemd-managed), system_monitor · verified by: TEST-HAL-060
  • FR-010
    The mobile application shall display the rover's current position overlaid on the aerial image.
    allocated to: mobile app, telemetry_publisher · verified by: VRF-FIELD
